# Botio React UI

This is a personal collection of shadcn components for Botio.

## Installation

Register the components in your `components.json` file.

```json
{
  "registries": {
    "@botio-ui": "https://github.com/alexbotio",
  }
}
```

## Usage

```bash
npx shadcn@latest add @botio-ui/shared/loading-screen
```
